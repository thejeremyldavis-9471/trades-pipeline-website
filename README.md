# The Trades Pipeline

> Built for the Trades. Wired for Growth.

Full-service digital marketing website for blue-collar trades businesses. Built with Next.js 15, TypeScript, Tailwind CSS, and Framer Motion.

## Pages

- `/` — Animated two-door gateway (B2C vs B2B)
- `/residential` — Full residential services & pricing page
- `/commercial` — B2B commercial lead generation page
- `/contact` — Strategy call booking form

## Stack

- **Next.js 15** (App Router, Turbopack)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion** (gateway animation)
- **shadcn/ui** components
- **Google Fonts** — Oswald (display) + Inter (body)

## Deploy

Deployed on [Vercel](https://vercel.com). Push to `main` triggers automatic deployment.
